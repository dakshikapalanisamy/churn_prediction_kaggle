# Customer Churn Prediction

Predict customer churn for a bank using machine learning.

## Dataset
- 10,000 bank customers
- Features: Credit score, age, tenure, balance, products, etc.
- Target: Exited (churned or not)

## Approach
1. Data exploration and cleaning
2. Encoding categorical variables
3. Feature engineering (age groups, balance indicators)
4. Trained 5 models: Random Forest, Logistic Regression, SVM, KNN, Gradient Boosting
5. Compared performance

## Results
- **Best Model:** [Gradient boost]
- **Accuracy:** [0.8675]
- **Key Churn Predictors:** Age, EstimatedSalary, CreditScore

## Tools
- Python, Pandas, Scikit-learn
- Matplotlib, Seaborn
